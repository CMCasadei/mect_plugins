ATCMspinbox

Este objeto permite el uso de un spinbox.

Para utilizarlo, siga los pasos siguientes:

  1. Arrastre el objeto a la página.

  2. Haga doble clic en el objeto.

  3. Defina el campo variable con el nombre de la variable Crosstable de la que desea leer/escribir. Pulse el botón “…” para seleccionar la variable deseada.

  4. Definir la variable visibilityVar con el nombre de la variable Crosstable de la que se desea leer el valor a utilizar para mostrar o no el objeto. Pulse el botón “…” para seleccionar la variable deseada.

  5. Ponga los valores deseados en las propiedades minimum, maximum, singleStep, value, decimals (use una variable REAL si desea tener decimales), prefix y suffix en la sección ”QDoubleSpinBox”.

  6. Marque la variable viewStatus si desea obtener información gráfica sobre el éxito o el fracaso de la escritura o la relectura.
