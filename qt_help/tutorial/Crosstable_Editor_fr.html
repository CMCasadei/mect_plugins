Éditeur de Crosstable

Une fois que vous avez créé le projet dans Qt, vous devez d’abord définir les variables que vous souhaitez utiliser pour créer votre application. Pour ce faire, on utilise la “Crosstable” qui génère la table de variables globales utilisée de manière partagée entre l'environnement IHM et l'API.

Pour ouvrir la “Crosstable” sélectionnez l’icône MectSuite:

Voici la “Crosstable”:

Dans la section “Goto Section” (voir rectangle jaune) il est possible de choisir le type de variables à afficher dans le tableau ci-dessous.

La zone sélectionnée en rouge est la partie que l'utilisateur peut définir pour créer et éditer ses propres variables de projet.

En passant la souris sur chaque champ, vous pouvez voir une brève description..

Pour insérer ou modifier une variable:

  1. Sélectionnez la ligne du tableau;

  2. Remplir / modifier les champs dans la zone sélectionnée en rouge;

  3. Sélectionnez la touche “Entrée” du clavier ou une ligne du tableau pour permettre l'insertion de la variable nouvellement créée / modifiée.

NOTE: dans le tableau, il est possible de se déplacer en utilisant les flèches comme s'il s'agissait d'un tableur.
