Structure de projet vide

Lorsque vous ouvrez un projet nouvellement créé, l'environnement Qt Creator présente tous les fichiers du projet dans le panneau de projet. Plus précisément, il y a une section consacrée à:

  • common → configurations communes (chargement des fichiers de langue)

  • languages → section pour un support multilingue

  • template → configurations génériques des différents chemins d'accès au système

  • Headers → en-têtes source par défaut du projet (extension de fichier.h)

  • Sources → C++ sources des composants par défaut du projet (extension de fichier.cpp)

  • Forms → sources graphiques dans lesquelles vous définissez l'aspect des pages (extension de fichier.ui)

  • Resources → Sources des icônes et images (seulement les fichiers.png)

La section la plus intéressante pour les utilisateurs d'ATCMPlugin est celle qui contient les Form du projet qui permet de personnaliser l'aspect graphique des pages par défaut et, dans les pages définies par l'utilisateur, la définition du comportement des objets graphiques insérés.
