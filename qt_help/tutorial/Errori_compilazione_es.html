Errores de compilación

Si se ha realizado al menos una de estas operaciones:

  • Actualización de la versión de MectSuite

  • Eliminación de páginas

  • Mover la carpeta del proyecto

  • Uso de Qt Linguist (no reporta ningún error de compilación pero no ve las traducciones)

es necesario “limpiar” el proyecto para evitar errores durante la compilación.

En particular, pulse Build → Run qmake

Espere a que la barra se ponga verde.

Luego pulse Build → Clean All

Espere a que la barra se ponga verde.

Pulse Build → Build All.
