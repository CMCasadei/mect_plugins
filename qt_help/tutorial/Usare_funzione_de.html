Eine Funktion verwenden

Um eine Funktion innerhalb eines Programms zu verwenden, wählen Sie sie auf der Registerkarte “Class” aus und ziehen sie in das Programm.

Für das Einfügen von Variablen siehen Sie Kapitel “Einfügen von Variablen in FUP – LD – SFC-Programme”.

Insbesondere können Sie mit dieser Funktion die Werte eines Signals mit den gewünschten Werten überholen. Das Eingangssignal ist beispielsweise ein analoges Spannungssignal (PLC_AnIn_1) mit einem Bereich von 0-10V (ISI - FSI) und Sie möchten dieses Signal überholen, um eine Unter-/Obergrenze von -200+300 (ISL-FSL) zu erreichen.

Die variable “Valore_ricondizionato” nimmt einen Wert zwischen -200 und +300 an, abhängig vom Wert der Variablen (PLC_AnIn_1).
