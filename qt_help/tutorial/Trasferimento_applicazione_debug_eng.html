Download of a graphic application in DEBUG mode

It is possible to run an application in debug mode, compiling it in debug mode (see red circle); select MECT icon and choose desired mode.

To insert a breakpoint in a row click on the grey zone on the left of the source row number: for example in Sources → page100.cpp at the function page::page100().

To run the sequence “Build, Deploy and Debug” press F5 or the “green triangle with ladybug” (see blue circle).

In the Debug menu (yellow circle) it is possible to control the application in “Debug” mode.
