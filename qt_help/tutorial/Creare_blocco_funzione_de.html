Einen Funktionsblock erstellen

Um einen Funktionsblock zuerstellen, z.B. für die Sprache Function Block Diagram (FBD), gehen Sie wie folgt vor:

  • Class auswählen

  • Wählen Sie Function Blocks im Baum aus und drücken Sie die rechte Maustaste

  • Auswählen und Drücken Insert Function Block

  • Wählen Sie im erscheinenden Fenster Function Block Diagram FB

  • Geben Sie den Namen ein, den Sie dem Funktionsblock geben möchten (z.B RegolazioneON_OFF)

  • Drücken Sie OK

Der Funktionsblock wurde so erstellt, indem er den Zweig Function Blocks des Projektbaums erweitert, sehen wir den Zweig.

Um den Funktionsblock einzugeben, wählen und doppelklicken Sie auf den Funktionsblocknamen. Das angezeigte Fenster besteht aus zwei Zonen:

  • Zone 1 zur Definition von Variablen

  • Zona 2 Code

Lassen Sie den Funktionsblock in einem ersten Schritt die in Crosstable definierten Variablen sehen. Positionieren Sie sich dazu mit der Maus in Zone1 (ändert die Hintergrundfarbe), unmittelbar nach dem Namen des Funktionsblocks, und geben Sie die Wörter: #import “Crosstable.gvl” ein, wie Sie in der Abbildung sehen können.

Die Ein- und Ausgangsvariablen (wenn es welche gibt, hinzufügen Sie "VAR_OUTPUT" und "END_VAR" ) müssen deklariert werden “END_VAR”).Sehen Sie sich beispielsweise das folgende Bild an:

In diesem Beispiel haben Sie beim Ziehen des Funktionsblocks in das Programm 2 Eingänge und 1 Ausgang.

Gehen Sie nun in Zone 2, um den Programmcode einzugeben.

Um Operatoren einzufügen, wählen Sie das Symbol Insert Programming Unit, in dem das Fenster erscheint, aus dem Sie die Operatoren der Bibliothek auswählen können. Durch Auswählen und Ziehen des erforderlichen Operators (z.B. des ADD-Operators, der zwei Variablen aufsummiert), wird er in die Zone 2 verschoben.

Um die Variablen mit dem Bediener zu verbinden, wählen Sie das Symbol, an dieser Stelle einfach die Verbindungen wie in einem normalen Editor für Schaltpläne herstellen. Zum Einfügen von Variablen siehen Sie Kapitel “Einfügen von Variablen in FUP – LD – SFC-Programme”.
