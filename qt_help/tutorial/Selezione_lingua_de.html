Sprache aus dem Gerät auswählen

Die Sprachauswahl am Gerät kann auf zwei verschiedene Arten erfolgen:

  1. Über die Standardseiten:

Wählen Sie auf der Menüseite “OPTIONS” → “LANGUAGE”

  1. Durch eine personalisierte Seite:

Wählen Sie eine ATCMbutton aus und wählen Sie mit der rechten Maustaste im erscheinenden Menü Go to slot… wählen Sie dann die Zeile pressed() und drücken Sie OK.

Das folgende Fenster kann erscheinen:

Ist dies der Fall, müssen die folgenden Schritte durchgeführt werden:

  • Machen Sie das Build des Projekt

  • Schließen und Öffnen das Projekt wieder

Dann erscheint eine Seite mit C++-Code:

void page100::on_atcmButton_3_pressed()

{

}


Fügen Sie den folgenden Code in die geschweiften Klammern der obigen Funktion ein:

if(translator->load(QString(":/translations/languages_it.qm"))) {

QSettings settings(CONFIG_FILE,QSettings::IniFormat);

settings.setValue("SYSTEM/language","it");

settings.sync();

}

else

{

QMessageBox::information(this,trUtf8("errore"),trUtf8("Caricamento lingua"));

}


Wiederholen Sie diese Schritte für alle Selektionen und ändern Sie nur die grün markierten Teile entsprechend.

Für Englisch ersetzen: :/translations/languages_en.qm

Für Englisch ersetzen: :/translations/languages_fr.qm

ACHTUNG: : Es ist notwendig, im ersten Teil der Datei folgende Zeile einzufügen: #include <QSettings>
