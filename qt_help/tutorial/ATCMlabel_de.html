ATCMlabel

Dieses Objekt ermöglicht die Visualisierung und Modifikation des Inhalts einer Variablen des Crosstable.

Um es zu verwenden, führen Sie die folgenden Schritte aus:

  1. Ziehen Sie das Objekt auf die Seite.

  2. Doppelklicken Sie auf das Objekt.

  3. Definieren Sie das Feld variable mit dem Namen der Crosstable-Variablen, aus der Sie lesen/schreiben möchten. Drücken Sie die Taste “…” um die gewünschte Variable auszuwählen.

  4. Definieren Sie die Variable visibilityVar mit dem Namen der Crosstable-Variablen, aus der Sie den Wert lesen möchten, der zur Anzeige des Objekts verwendet werden soll oder nicht. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  5. Kreuzen Sie die Variable viewStatus an, wenn Sie eine grafische Rückmeldung über den Erfolg oder Misserfolg des Schreibens oder Nachlesens wünschen.

Wenn das Label gedrückt wird, erscheint eine Tastatur, in der Sie den gewünschten Wert eingeben können.
