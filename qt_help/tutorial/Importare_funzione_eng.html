Import a function block or a function

It is possible to import a function / function block already defined in another project:

  • Select “File” tab

  • Select “Source Files” and press right mouse key

  • Select “Add File(s)”

  • Select the function / function block to import

  • Select “Open”

The imported function / function block is visible in the “Class” tab in the “Functions” or “Function Blocks” item. In this case a function is imported.
