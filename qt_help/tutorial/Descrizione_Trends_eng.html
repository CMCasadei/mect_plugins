Description of Trend fields

Field

Description

Orientation

Trend orientation respect time axis:

- “Landscape” → horizontal;

- “Portrait” → vertical.

Visible

Trace visibility in the trend page.

Variable name

Variable name to show in the trend page.

Color

Color of the trace in the trend page.

Min. value

Minimum value of the trend on Y axis.

Max. value

Maximum value of the trend on Y axis.

Description

Description of the trace.
