Das Kompiliern

Das Kompiliern eines Projekts kann auf zwei verschiedene und völlig gleichwertige Arten erfolgen.

Es ist möglich, über die Schaltfläche im linken Bereich in Bezug auf die Werkzeuge, die für das Projekt verwendet werden können, zu kompilieren. Alternativ können Sie auch aus dem Menü BUILD den Eintrag “Build Project PPP.pro” auswählen, wobei PPP das von Ihnen durchgeführte Projekt ist. Beide Möglichkeiten sind in der folgenden Abbildung dargestellt.
