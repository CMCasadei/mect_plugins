Seleccionar el idioma desde el dispositivo

La selección del idioma en el dispositivo se puede hacer de dos maneras diferentes:

  1. A través de las páginas por defecto:

Desde la página de menú seleccione “OPTIONS” → “LANGUAGE”

  1. A través de una página personalizada:

Seleccione un ATCMbutton y, con el botón derecho del ratón, seleccione Go to slot… en el menú que aparece; a continuación, seleccione la línea pressed() y pulse OK.

Puede aparecer la siguiente ventana:

Si este es el caso, se deben realizar los siguientes pasos:

  • Hacer el Build del proyecto

  • Cerrar y reabrir el proyecto

Entonces aparecerá una página con el código C++:

void page100::on_atcmButton_3_pressed()

{

}


Inserte el siguiente código en los corchetes de la función anterior:

if(translator->load(QString(":/translations/languages_it.qm"))) {

QSettings settings(CONFIG_FILE,QSettings::IniFormat);

settings.setValue("SYSTEM/language","it");

settings.sync();

}

else

{

QMessageBox::information(this,trUtf8("errore"),trUtf8("Caricamento lingua"));

}


Repita estos pasos para todas las selecciones y cambie sólo las partes escritas en verde apropiadamente.

Para inglés reemplazar: :/translations/languages_en.qm

Para el francés reemplazar: :/translations/languages_fr.qm

ATENCIÓN: es necesario insertar en la parte inicial del fichero la siguiente línea: #include <QSettings>
