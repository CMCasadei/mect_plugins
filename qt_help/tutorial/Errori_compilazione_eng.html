Build errors

If at least one of these operations was performed:

  • Update Plugin

  • Removing page

  • Moving the project folder

  • Qt Linguist use (it does not report any compilation error but translations are not visible)

you need to make a “cleaning” of the project to avoid errors during compilation.

Press Build → Clean All

Wait until the bar turns green.

Then press Build → Run qmake

Wait until the bar turns green.

Press Build → Build All.
