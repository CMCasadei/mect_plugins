ATCMspinbox

Object to use a spin box.

To use follows the steps:

  1. Draw object on page.

  2. Double click on object.

  3. Define variable field with Crosstable variable name to read/write. Press “…” button to select desired variable.

  4. Define visibilityVar variable with Crosstable variable to read value to show or not the object. Press “…” button to select desired variable.

  5. Fill with desired values the properties minimum, maximum, singleStep, value, decimals (use REAL variable type for decimal use), prefix and suffix in the section ”QDoubleSpinBox”.

  6. Check viewStatus variable to have a graphic feedback for reading/writing.
