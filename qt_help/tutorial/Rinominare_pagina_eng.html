Rename an existing page

In order to avoid malfunctions in the project, it is recommended not to rename an existing page. If you need to do it please follow the following steps accurately:

  1. Create a new page with the name you want;

  2. On the page that you would like to rename: select all items. Edit → Select All

  1. Copy the elements: Edit → Copy

  2. Go to the new page created and select Edit → Paste

  3. Remove the page that you wanted to remove like chapter “Remove an existing page”.
