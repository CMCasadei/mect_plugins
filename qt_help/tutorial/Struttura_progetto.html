Struttura progetto vuoto

All’apertura di un progetto appena creato l’ambiente Qt Creator presenta nel pannello relativo al progetto tutti i file di cui si compone il progetto stesso. Nello specifico si ha una sezione dedicata a:

  • common → configurazioni comuni (caricamento file di lingue)

  • languages → sezione per il supporto multilingue

  • template → configurazioni generiche dei vari path di sistema

  • Headers → intestazioni dei sorgenti di default del progetto (file con estensione .h)

  • Sources → sorgenti C++ delle componenti di default del progetto (file con estensione .cpp)

  • Forms → sorgenti grafici nei quali si definisce l’aspetto delle pagine (file con estensione .ui)

  • Resources → sorgenti delle icone e immagini (solamente file .png)

La sezione di maggior interesse per gli utenti ATCMPlugin è quella contente i Form del progetto che consente la personalizzazione dell’aspetto grafico delle pagine di default e, nelle pagine definite dall'utente la definizione del comportamento degli oggetti grafici inseriti.
