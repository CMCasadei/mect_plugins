ATCManimation

Dieses Objekt ermöglicht die Visualisierung einer Reihe von Frames im png-Format, die dem Wert zugeordnet sind, der von einer ganzzahl-Variablen des Crosstable.

Um es zu verwenden, führen Sie die folgenden Schritte aus:

  1. Ziehen Sie das Objekt auf die Seite.

  2. Doppelklicken Sie auf das Objekt und füllen Sie die Mapping-Tabelle aus, wobei Sie beachten müssen, dass “Value” Sie in der Spalte den Wert eingeben müssen, der von der SPS gelesen wird, und in der Spalte “Picture” das Bild, das auf der Schnittstelle angezeigt wird.

  3. Definieren Sie das Feld variable mit dem Namen der Crosstable-Variablen, aus der Sie lesen/schreiben möchten. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  4. Definieren Sie die variable visibilityVar mit dem Namen der Variablen Crosstable, aus der Sie den Wert lesen möchten, der zur Anzeige des Objekts verwendet werden soll oder nicht. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.
