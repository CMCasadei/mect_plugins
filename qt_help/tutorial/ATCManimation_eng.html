ATCManimation

Object to show a series of frame in png format linked to an integer Crosstable value.

To use follows the steps:

  1. Drag object on page.

  2. Double click on object and compile the table. In the “Value” column write the value to pass to PLC and in the “Picture” column write the value for the interface.

  3. Define variable field with Crosstable variable name to read/write. Press “…” button to select desired variable.

  4. Define visibilityVar variable with Crosstable variable to read value to show or not the object. Press “…” button to select desired variable.
