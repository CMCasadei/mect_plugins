S / F / V / X type update

To store a variable you must define it as “S”, “F”, “V”, and “X” update type variable and it is necessary to write the “logStart()” function inside the desired code or in the “void setup(void)” function of the “automation.cpp” page in the Edit → Sources section.

Moreover it is possible to stop, reset and instantaneously force the store process using the function logStop(), logReset() and logShot().
