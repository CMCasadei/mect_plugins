Añadir un nuevo archivo de idiomas

Para añadir un nuevo archivo de idioma es necesario:

  • Vaya al árbol del proyecto en la sección de idiomas.

  • Abra el archivo “languages.pri” y añada el nombre del archivo de traducción del idioma deseado recordando que el archivo debe llamarse languages_*.ts donde se debe colocar el código del idioma deseado en lugar de * (por ejemplo: fr → francés). Los códigos a utilizar son visibles en esta página web: wiki.freepascal.org/Language_Codes

  • ATENCIÓN: entre el nombre de un fichero y el otro hay que insertar el símbolo “ \ ”.

  • Compilar para crear el archivo de traducción en el sistema de archivos (creación de archivos: *.qm y *.ts).

  • Vaya al árbol del proyecto en la sección “common” → “Resources”.

  • Añada al archivo de recursos “languages.qrc” el correspondiente archivo *.qm creado al compilar el paso anterior.

  • Guardar todos: File → Save “resources.qrc”.
