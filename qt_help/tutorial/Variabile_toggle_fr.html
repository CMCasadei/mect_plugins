Variable de commutation

Pour utiliser le bouton pour effectuer un changement d'état d'une variable de commutation, procédez comme suit:

  1. Faites glisser l'objet sur la page.

  2. Double-cliquez sur le bouton.

  3. Définissez la variable d'état StatusVar avec le nom de la variable Crosstable que vous voulez utiliser pour changer la valeur en fonction de l'état du bouton. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

    NOTE: si vous voulez changer l'état seulement jusqu'à ce que le bouton soit pressé, vérifiez que l'option checkable NON SOIT cochée; si vous voulez changer l'état jusqu'à la prochaine fois que vous pressez, vérifiez que l'option checkable dans la section QAbstractButton SOIT cochée.

NOTE: si la valeur de la variable de référence change depuis le PLC(automate), le bouton prendra l'état de la variable elle-même. Si la variable est forcée à 1, alors le bouton sera pressé et effectuera toute opération associée. Si la variable est forcée à 0, le bouton est relâché.
