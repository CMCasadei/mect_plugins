Timings

La ventana di “Timing” da una idea general del tiempo necesario para permitir un correcto intercambio de valores entre variables con el protocolo RTU.

Para iniciar la ventana “Timing”, véase la figura siguiente:


La interfaz es la siguiente:

La vista de árbol es similar a la vista de pantalla “Devices”, pero las variables de la columna Name están agrupadas por:

  • Panel → Device (interfaz de comunicación) → Priority → Node → Block → Variables

La columna Info contiene la misma información que la ventana de la pantalla “Devices”.

La columna Timings indica, para los protocolos RTU, el tiempo necesario para leer el elemento seleccionado (Device, Priority, Node, Block).
