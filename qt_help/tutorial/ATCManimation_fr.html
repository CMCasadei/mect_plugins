ATCManimation

Cet objet permet la visualisation d'une série de trames au format png associée à la valeur assumée par une variable entière de la Crosstable.

Pour l'utiliser, suivez les étapes ci-dessous:

  1. Faites glisser l'objet sur la page.

  2. Double-cliquez sur l'objet et remplissez la table de correspondance en vous rappelant que dans la colonne “Value” vous devez entrer la valeur qui sera lue par l'automate(PLC) et dans la colonne “Picture” vous devez entrer l'image qui sera affichée sur l'interface.

  3. Définissez le champ variable avec le nom de la variable Crosstable à partir de laquelle vous voulez lire/écrire. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  4. Définissez la variabile visibilityVar avec le nom de la variable Crosstable à partir de laquelle vous voulez lire la valeur à utiliser pour afficher ou non l'objet. Appuyez sur le bouton “…” pour sélectionner la variable désirée.
