Trends Editor functionality

  • Save: save actual setup.

  • Save as: save actual setup in a desired folder. Saving is possible only in the “config” folder of the project.

    WARNING: the name of the file must be: trendxy where xy are numbers, in order to recall the file directly in the property “pageName” of an ATCMbutton of Qt Creator.

  • Check: check the correctness of setup. Led on the key changes its color according to the state:

    • Red: wrong setup (application not working);

    • Green: check OK.

  • Import Trend: loads the file “trend….csv” of another project.

  • Exit: close program and ask for saving last setup.

  • Delete file: delete trend file in the project. If there is only one trend file, the delete is not possible.
