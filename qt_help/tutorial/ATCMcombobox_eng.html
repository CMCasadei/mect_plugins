ATCMcombobox

Object to use a combo box.

To use follows the steps:

  1. Draw object on page.

  2. Double click on object and compile the table. In the Value column write the value to pass to PLC and in the Label column write the value for the interface.

  3. Define variable field with Crosstable variable to write/read. Press “…” button to select desired variable.

  4. Define visibilityVar variable with Crosstable variable to read value to show or not the object. Press “…” button to select desired variable.

  5. Check viewStatus variable to have a graphic feedback for reading/writing.
