Uso del convertidor USB/RS232 en la HMI

En MectApps hay disponible un ejemplo para realizar lecturas y escritura utilizando un convertidor USB/RS232. Este proyecto se llama “USB_RS232” y se encuentra bajo la siguiente ruta: MectApps → AnyTPAC.

La lógica del programa se encuentra en la página “automation.cpp”.

El ejemplo se realiza utilizando un driver Prolífico.
