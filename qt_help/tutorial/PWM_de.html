Verwendung von PWM

In verschiedenen Geräten ist es möglich, die analogen Ausgänge im PWM mit einer festen Frequenz von 250Hz zu verwenden. Die Konfiguration kann direkt aus der SPS oder über die grafische Oberfläche erfolgen.

Führen Sie die folgenden Schritte aus, um den Analogausgang (mit der Sprache ST ) zu konfigurieren, z.B. im TPAC1007_04_AC:

  1. Konfigurieren Sie den analogen Spannungsausgang: PLC_AnOutConf_1 := 2;

  2. Stellen Sie die Amplitude ein, z.B. 6V: PLC_AnOut_1 := 600;

  3. Konfigurieren Sie den Analogausgang im PWM: PLC_AnOutConf_1 := 3;

  4. Stellen Sie das Tastverhältnis ein, z.B. 40%: PLC_AnOut_1 := 40;

Die Punkte 3 und 4 müssen in einer zweiten Zykluszeit der SPS durchgeführt werden. Um den Tastverhältniswert zu ändern, wiederholen Sie Schritt 4.

WARNUNG: Wenn dieser Mechanismus über das HMI ausgeführt wird, müssen aufgrund der Darstellung der beiden Nachkommastellen des Ausgangs unterschiedliche Werte eingegeben werden. In Schritt 2 setzen Sie 6, während Sie in Schritt 4 den Wert 0,4 eingeben.
