HMI

HMI creation is done in the QtCreator tool of Qt customized by ATCMPlugin.

ATCMPlugin is a collection of graphics objects that allows a creation of an HMI interface without writing code.

Graphics objects are based on Qt 4.8.5 libraries (http://qt-project.org/doc/qt-4.8) written in C++.

Variables used in ATCMPlugin must be explicitly present in the “Crosstable Editor”.

In the Crosstable all variables needed in the application are present.

Qt installation available for MECT products allows compilation in DEBUG and RELEASE mode. In the RELEASE mode you can not debug step by step and as a result the executable appears to be very small in size.
