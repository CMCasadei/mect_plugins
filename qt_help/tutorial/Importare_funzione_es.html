Importación de un bloque de función / función

Es posible importar una función / un bloque de funciones previamente utilizado en otro proyecto. Para hacer esto, tiene que :

  • Seleccionar la pestaña “File”

  • Seleccione “Source Files” y pulsar el botón derecho del ratón

  • Seleccionar “Add File(s)”

  • Seleccionar la función / el bloque de funciones que desea importar al nuevo proyecto.

  • Seleccionar “Open”

La función/ el bloque de funciones recién importado se muestra en la pestaña “Class” bajo “Functions” o “Function Blocks”. En este caso, se ha importado una función.
