Insertar variables en programas ST

Para insertar variables rápidamente, sitúe el cursor en la línea deseada y pulse F4 o Select Variable con el botón derecho del ratón.

Puede introducir la primera letra de la variable deseada para tener un filtro en la búsqueda.

Como alternativa, para insertar variables, funciones o bloques de función en el programa, siga los pasos que se indican a continuación:

  • Seleccione “View” → “POU/Variable Selection” → “Variable Selection&rdquo (o pulse el botón derecho del ratón dentro de un programa);

    • Seleccione la variable deseada (también puede utilizar filtros) y pulse “Apply”

Para insertar una función / bloque de función:

  • Seleccione “View” → “POU/Variable Selection” → “POU Selection”

    • Seleccione la función / el bloque de función deseada e pulse “Apply”

Obviamente, los nombres de las variables, funciones y bloques de funciones se pueden escribir directamente sin seguir el procedimiento descrito anteriormente.
