Mise à jour de type S / F / V / X

Pour mémoriser une variable je dois la définir avec mise à jour de type “S”, “F”, “V”, et “X” et je dois utiliser la fonction “logStart()” soit dans le code désiré soit dans la fonction “void setup(void)” de la page “automation.cpp” dans Edit → Sources.

Vous pouvez également arrêter, réinitialiser et forcer instantanément la mémorisation en utilisant les fonctions logStop(), logReset() et logShot().
