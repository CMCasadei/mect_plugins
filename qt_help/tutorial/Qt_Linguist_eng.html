QtLinguist use

Linguist tool is a graphic front end to compile translation tables. Tolls is as in the following figure. Context section contains all the tree of the pages of the project for which create the translation. If you click on a page of the project in the string section the list of translations to compile for the selected page appears. Incomplete translations are marked with and a complete translation is marked with .

As soon as elements of “String” section are selected in the “Source text” the actual text appears without translation. In the “…translation” field (bottom page) it is possible to insert the desired translation. To terminate the input of the translation click on the key or CTRL + enter; to indicate that the translation is complete; system points now on the next “Source text” of the “String” section.

At the end it is necessary to save the file, using the save key.

WARNING: if a translation is inserted but the key is not pressed, on the panel the “Source text” is displayed instead of the translation text.

WARNING: do not translate the names of the variables of the crosstable.
