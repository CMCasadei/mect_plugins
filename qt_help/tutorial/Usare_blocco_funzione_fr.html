Utiliser un bloc function

Pour utiliser un bloc fonction dans un programme, sélectionnez le bloc et faites-le glisser dans le programme.

Plusieurs instances d'un bloc fonction peuvent être utilisées dans un programme. Pour cette raison, chaque bloc fonction doit être déclaré.

Sélectionnez le bloc fonction dans le programme, double-cliquez sur la partie supérieure (partie sombre) et écrivez le nom de l'instance que vous voulez donner à ce bloc fonction.



→

Ensuite, vous devez déclarer la nouvelle instance du bloc fonction (à l'intérieur entre VAR et END_VAR).

Pour ce faire, vous pouvez faire glisser les blocs fonctionnels présents dans la “Class”, directement dans le VAR et END_VAR, n'ayant qu'à renommer le bloc (par exemple “Regolazione_1”) .

Pour l'insertion de variables, voir le chapitre “Insérer des variables dans les programmes FBD – LD – SFC”.
