Modèles TPLC050 / TPLC100

Le tableau suivant présente les principales variables du produit. Ces variables sont présentes dans la Crosstable et sont visibles depuis la ligne 5300 de “Crosstable Editor” Par exemple, si vous voulez contrôler à partir de l'IHM l'action d'un bouton ATCM qui avec la pression change la sortie numérique 1 ou si vous voulez initialiser à partir de PLC, la sortie numérique 1 à une valeur, vous devez utiliser directement la variable PLC_DigOut_1.

Pour insérer les variables de nœuds supplémentaires (MPNC020 / MPNC030 / MPNC035) dans le projet TPLC050, cliquez sur “Paste TPLC050 Modules”, voir “Foncionnalité Crosstable Editor”
