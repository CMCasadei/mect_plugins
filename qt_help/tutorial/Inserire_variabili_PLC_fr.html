Insérer des variables dans les programmes FBD - LD - SFC

Pour insérer les variables dont on a besoin, il faut faire glisser l'icône Insert Var Read dans la fenêtre de la zone 2.

Pour associer rapidement la variable désirée, sélectionnez l'objet et appuyez sur F4 ou Select Variable avec le bouton droit de la souris.

Vous pouvez entrer la première lettre de la variable désirée pour avoir un filtre dans la recherche.

Alternativement , double-cliquez sur la variable entrée dans la fenêtre et changez le nom en entrant la variable à laquelle on veut l'associer(par exemple, une variable dans la “Crosstable.gvl”)ou sélectionnez la variable insérée, sélectionnez l'icône Insert Programming Unit et double cliquez sur la variable que vous voulez associer.
