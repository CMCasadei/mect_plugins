Connexion locale

Pour pouvoir télécharger l'application de votre PC vers l'appareil Mect , vous devez configurer votre ordinateur avec le sous-réseau 5 ou modifier le sous-réseau de l'appareil Mect et le configurer de la même manière que votre PC.

Pour modifier le sous-réseau de votre PC, vous pouvez suivre les instructions de Windows sous le lien suivant: (configurez le “Protocole Internet Version 4(TCP/IPv4)”)

https://support.microsoft.com/en-us/help/15089/windows-change-tcp-ip-settings

Suivez les instructions du guide, y compris le point 7 (appuyez sur “Avanceés…”) et sélectionnez le bouton “Ajouter…” en entrant les paramèters suivants:

  • Adresse IP : 192.168.5.100 (doit être différente de celle du panneau opérateur)

  • Masque de sous-réseau: 255.255.255.0


Pour modifier le sous-réseau de l'appareil Mect, appuyez sur le bouton situé en haut à droite(présent dans l'application native de l'usine où le nom du produit acheté est écrit) puis appuyez sur: OPTIONS → NETWORK → ETHERNET. Changez l'adresse IP et sauvegardez le changement.

Pour modifier le sous-réseau dans votre application, vous devez utiliser un “ATCMbutton” e insérer dans la propriété; “pageName” le nom de la page de “librairie” net_conf
