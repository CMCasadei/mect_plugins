Page change

To change a page by a button follows these steps:

  1. Drag the object on the page.

  2. Select button and write in the page_Name property (right side) the page name where to go (if home page is the desired page write HOME, if the last page is desired write BACK). If a system page is desired, see paragraph “Library pages”.

  3. Check remember property to add the destination page in the page history in order to use it with BACK functionality.
