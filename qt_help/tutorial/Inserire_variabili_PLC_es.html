Insertar variables en programas FBD - LD - SFC

Para insertar las variables que necesitamos, arrastre el icono Insert Var Read en la ventana de la zona 2.

Para asociar rápidamente la variable deseada, seleccione el objeto y pulse F4 o Select Variable con el botón derecho del ratón.

Puede introducir la primera letra de la variable deseada para tener un filtro en la búsqueda.

Alternativamente, haga doble clic sobre la variable introducida en la ventana y cambie el nombre introduciendo la variable a la que queremos asociarla (por ejemplo, una variable en el “Crosstable.gvl”) o seleccione la variable que acaba de insertar, seleccione el icono Insert Programming Unit y haga doble clic sobre la variable que desea asociar.
