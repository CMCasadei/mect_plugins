Add a new language file

To add a new language file do:

  • Go to the project tree in the language section.

  • Open “languages.pri” file and add the desired name file of the desired language, the name must be languages_*.ts where * is the code of the desired language (ex: fr → french). The codes to be used are displayed on this web page: wiki.freepascal.org/Language_Codes

  • WARNING: between a name and another put “ \ ” char.

  • Compile to create the translation file on system file (creation of files: *.qm and *.ts).

  • Go to the project tree in the “common” → “Resources” section.

  • Add in the resource file “languages.qrc” the relevant file *.qm created in the previous compilation.

  • Save all: File → Save “resources.qrc”.
