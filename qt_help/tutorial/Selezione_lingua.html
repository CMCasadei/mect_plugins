Selezionare la lingua dal dispositivo

La selezione della lingua sul dispositivo può avvenire in due modi differenti:

  1. Attraverso le pagine di default:

Dalla pagina di menu selezionare “OPTIONS” → “LANGUAGE”

  1. Attraverso una pagina personalizzata:

Selezionare un ATCMbutton e con il tasto destro del mouse, selezionare Go to slot… nel menu che appare; quindi selezionare la riga pressed() e premere OK.

Potrebbe apparire la seguente finestra:

Se così fosse si devono eseguire i seguenti passaggi:

  • Fare il Build del progetto

  • Chiudere e riaprire il progetto

Successivamente apparirà una pagina con codice C++:

void page100::on_atcmButton_3_pressed()

{

}


Inserire il seguente codice all’interno delle parentesi graffe della suddetta funzione:

if(translator->load(QString(":/translations/languages_it.qm"))) {

QSettings settings(CONFIG_FILE,QSettings::IniFormat);

settings.setValue("SYSTEM/language","it");

settings.sync();

}

else

{

QMessageBox::information(this,trUtf8("errore"),trUtf8("Caricamento lingua"));

}


Ripetere queste operazioni per tutte le selezioni e cambiare solo le parti scritte in verde in modo opportuno.

Per inglese sostituire: :/translations/languages_en.qm

Per francese sostituire: :/translations/languages_fr.qm

ATTENZIONE: è necessario nella parte iniziale del file inserire la seguente riga: #include <QSettings>
