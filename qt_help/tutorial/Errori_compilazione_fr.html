Erreurs de compilation

Si au moins une de ces opérations a été effectuée:

  • Mise à jour des plugins

  • Suppression d'une page

  • Déplacer le dossier de projet

  • Utilisation de Qt Linguist (ne signale aucune erreur de compilation mais vous ne voyez pas les traductions)

vous devez “nettoyer” le projet pour éviter les erreurs lors de la compilation.

En particulier, appuyez sur Build → Clean All

Attendez que le bar devienne vert.

Puis appuyez sur Build → Run qmake

Attendez que le bar devienne vert.

Appuyez sur Build → Build All.
