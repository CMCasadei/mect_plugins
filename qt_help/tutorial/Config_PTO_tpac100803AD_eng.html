PTO configuration

TPAC1008 / TPLC100: before configuring, wait until the variable “CH0_01_NODERUN” is at “TRUE”.


Variable

Configuration

PLC_Pwm_Steps_..

Number of steps to perform the PTO.

If PLC_Pwm_Steps_..= 0 the PTO counts indefinitely.

PLC_Pwm_Freq_..

Setting the PTO frequency

If PLC_Pwm_Freq_..= 0 the PTO output behaves like a standard output.
