Funzionalità Trends Editor

  • Salva: salvataggio delle impostazioni attuali.

  • Salva In: salvataggio delle impostazioni attuali in una locazione desiderata. Il salvataggio dei file può avvenire solamente nella cartella “config” del progetto in corso.

    ATTENZIONE: bisogna chiamare i file: trendxy dove xy sono dei numeri, affinché il file possa essere richiamato direttamente dalla proprietà “pageName” di un ATCMbutton di Qt Creator.

  • Controlla: il tasto verifica la correttezza delle impostazioni inserite. Il led presente sul tasto cambia colore in funzione dello stato:

    • Rosso: impostazioni da controllare e potenzialmente non corrette (malfunzionamento dell’applicazione);

    • Verde: controllo avvenuto con esito positivo.

  • Importa Trend: permette il caricamento del file “trend….csv” presente all’interno di un altro progetto.

  • Esci: chiusura del programma in uso e nel caso siano state variate delle impostazione rispetto l’ultimo salvataggio chiede se si desidera salvare l’attuale configurazione.

  • Cancella file: effettua la cancellazione dei file di Trend all’interno del progetto. Se nel progetto è presente solamente un file, ad esempio trend1.csv, tale operazione viene negata.
