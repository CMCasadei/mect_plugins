Création d'un nouveau projet

Pour créer un nouveau projet graphique, il suffit de lancer simplement Qt Creator et selectionnez l'onglet “Develop”.

suivez les étapes suivantes:

  1. Selectionnez “Create Project”.

  2. Lorsque la fenêtre dans la figure s'ouvre, choisissez “Mect Projects”, le produit et cliquez sur “Choose”.

  1. Inserez le nom du produit, sans espaces et sans caractères spéciaux sauf underscore ‘_’.

  2. ATTENTION: de plus, le chemin du dossier de projet ne doit pas contenir d'espaces.

  3. Selectionnez “Next” à la demande de “kit selection”.

  4. Selectionnez l'orientation du produit pour lequel vous voulez faire le projet.

  1. Dans la fenêtre “Project Management” cliquez sur “finish”.

ATTENTION: dans le dossier de projet se trouvent les dossiers “icons” et “fonts” où vous pouvez insérer vos images / polices pour personnaliser votre application
