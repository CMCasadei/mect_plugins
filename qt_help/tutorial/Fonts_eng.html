Fonts

During installation of MectSuite following fonts are installed:

  • DejaVu Sans

  • DejaVu Serif

We recommend to use these fonts in the applications because other fonts are not correctly shown on panel.

To add a font on target it is necessary to include it in the project, following the procedure in the chapter “Add images / fonts” in the resource file.

WARNING: it is possible to add only fonts with .ttf extension.
