Connectivity

It is possible to connect the panel through the net LAN, Wi-Fi and Mobile. Lan connectivity is available directly on Ethernet connector and the Mobile and Wi-Fi connection is available using an USB dongle.

WARNING: only dongles sold by Mect are tested and guaranteed.

System supports also a VPN connection. With these elements it is possible to create a monitoring structure, for example sending LOGS to an external server or configuring the system it is possible to connect to the panel via VNC.
