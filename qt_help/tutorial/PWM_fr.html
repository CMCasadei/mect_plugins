Utilisation du PWM

Dans différents appareils, il est possible d'utiliser les sorties analogiques en PWM à une fréquence fixe de 250 Hz. La configuration peut se faire directement depuis le PLC(API) ou par interface graphique.

Pour configurer la sortie analogique (en utilisant le langage ST du PLC ), par exemple dans le TPAC1007_04_AC, procéder comme suit:

  1. Configurer la sortie analogique en tension: PLC_AnOutConf_1 := 2;

  2. Régler l'amplitude, par exemple 6V: PLC_AnOut_1 := 600;

  3. Configurer la sortie analogique en PWM : PLC_AnOutConf_1 := 3;

  4. Régler le rapport cyclique, par exemple 40% : PLC_AnOut_1 := 40;

Les points 3 et 4 doivent être effectués dans un deuxième temps de cycle de l'automate plc. Pour modifier la valeur du rapport cyclique, répéter l'étape 4.

ATTENTION: si ce mécanisme est réalisé par HMI, des valeurs différentes doivent être entrées en raison de la représentation des 2 décimales de la sortie. A l'étape 2, vous mettrez 6 alors qu'à l'étape 4, vous mettrez la valeur 0,4.
