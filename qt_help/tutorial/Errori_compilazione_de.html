Kompilierungsfehler

Wenn mindestens einer dieser Vorgänge durchgeführt wurde:

  • Plugin-Aktualisierung

  • Seitenentfernung

  • Verschieben des Projektordners

  • Verwendung von Qt Linguist (meldet keinen Kompilierungsfehler, aber Sie sehen die Übersetzungen nicht)

müssen Sie das Projekt “bereinigen”, um Fehler bei der Kompilierung zu vermeiden.

Drücken Sie insbesondere Build → Clean All

Warten Sie, bis der Balken grün wird.

Dann drücken Sie Build → Run qmake

Warten Sie, bis der Balken grün wird.

Drücken Sie Build → Build All.
