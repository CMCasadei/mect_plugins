Editor de Crosstable

Una vez que haya creado el proyecto en Qt, primero debe definir las variables que desea usar para hacer su aplicación. Para hacer esto, se utiliza la “Crosstable” que genera la tabla de variables globales que es usada de forma compartida entre el entorno HMI y PLC.

Para abrir la “Crosstable” seleccione el icono MectSuite:

Aquí está la “Crosstable”:

En la sección “Goto Section” (ver rectángulo amarillo) es posible elegir el tipo de variables a visualizar en la siguiente tabla.

El área seleccionada en rojo es la parte que el usuario puede configurar para crear y editar sus propias variables de proyecto.

Pasando el mouse sobre cada campo puede ver una breve descripción..

Para insertar o modificar una variable:

  1. Seleccione la fila de la tabla;

  2. Rellene / modifique los campos en el área seleccionada de rojo;

  3. Seleccione la tecla “Enter” del teclado o cualquier fila en la tabla para permitir la inserción de la variable recién creada / modificada.

NOTA: dentro de la tabla es posible moverse usando las flechas como si fuera una hoja de cálculo.
