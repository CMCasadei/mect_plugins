Modify/add a Task

For expert users it is possible to change Task priority and cycle time.

  • To change priority select Task and right click on “Properties”; change priority value in “Priority” field.

  • To add a new task select Resource1 and right click , press “Insert Task”. Now insert in the field:

    • “Name”: Task name;

    • “Priority”: priority;

    • “Interval”: task cycle time; insert T#xxms where xx is cycle time.

Then insert desired program into the created task.
