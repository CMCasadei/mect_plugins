Build and download of the automation

At the end of the program development, compile it to check errors in the code. Select “Build” → “Build All” or press F7.

After the compile step it is necessary to send the project to the panel.

To download the compiled project to the panel follow these steps:

  • Select Target tag then press mouse right key and select “TPAC= Resource{localhost}”. In the windows that appears select Proprierties.

  • Setup the panel IP address in the Control Address of TCP/IP tag, then press OK (factory default is: 192.168.5.211)

  • Select Instance tag

  • Select TPAC = Resource1 {localhost} then press mouse right key and select Flash (and Download)

  • At the end of download, in the bottom of the windows the message “Download Finished” is shown. If an error windows appears try to select again “Flash (and Download)”.

  • If a debug is not necessary disconnect from the panel, to do this select TPAC = Resource1 {localhost} and press with the right mouse key Disconnect.
