Empty project structure

If you open a newly created project, QT creator environment shows in the panel all the files that belong to the the project. In particular:

  • common → common configuration (language file upload)

  • languages → multilingual support section

  • template → generic configuration of system path

  • Headers → default source (.h files)

  • Sources → C++ default source code (.cpp files)

  • Forms → graphic source code where you define the page layout (.ui files)

  • Resources → icons and image source (file .png only)

The most interesting section for ATCMPlugin users is that containing Form of the project to customize the graphic aspect of the default pages and, in the user pages, the behaviour of graphics object inserted.
