Timings

The “Timings” window gives a general idea about the time necessary to allow a correct exchange of values between variables with RTU protocol.

To start “Timings” see following figure:


Here the interface:

The tree view is similar to the “Devices” screen view, but the variables in the Name column are grouped by:

  • Panel → Device (communication interface) → Priority → Node → Block → Variables

The Info column contains the same information as the “Devices” screen.

The Timings column indicates, for the RTU protocols, the time required to read the selected element (Device, Priority, Node, Block).
