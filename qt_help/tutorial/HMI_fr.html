IHM

La programmation de l'interface utilisateur IHM se fait dans l'environnement QtCreator de Qt personnalisé via ATCMPlugin.

ATCMPlugin ATCMPlugin est un ensemble d’objets graphiques vous permettant de créer des interfaces utilisateur à partir du panneau opérateur,ce qui rend l’écriture de code minimale et souvent absente.

Les objets graphiques sont basés sur les bibliothèques Qt 4.8.5 (http://qt-project.org/doc/qt-4.8) qui sont à leur tour implémentées en C ++.

Les variables utilisées par ATCMPlugin doivent être explicitement listées dans “Crosstable Editor”.

La Crosstable contient toutes les variables utilisées pour créer votre application.

L'installation de Qt disponible avec les produits MECT permet la compilation du projet en mode DEBUG et en mode RELEASE. En mode RELEASE, il n’est pas possible de déboguer étape par étape; l’exécutable est donc de très petite taille, alors qu’en mode DEBUG, il est possible de déboguer.
