PWM output

In many devices you can use the PWM analog outputs with a fixed frequency of 250Hz. The configuration can be done directly by plc or HMI.

Follow these steps to configure the analog output (use of the plc ST language), for example in TPAC1007_04_AC:

  1. Set the analog output as voltage: PLC_AnOutConf_1 := 2;

  2. Set the amplitude, for example 6V: PLC_AnOut_1 := 600;

  3. Configure the analog output as PWM: PLC_AnOutConf_1 := 3;

  4. Set the duty cycle, for example 40%: PLC_AnOut_1 := 40;

The steps 3 and 4 must be in a second cycle of the plc time. Repeat step 4 for change the duty cycle value.

WARNING: if this mechanism is done by HMI you must enter different values (representation of two decimal value in output). In step 2 it will put 6 and in step 4 the value 0.4.
