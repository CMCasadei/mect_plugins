New project creation

To create a new project run Qt Creator and select Tab “Develop”.

Follow the steps:

  1. Select “Create Project”.

  2. When the window in the figure opens, choose “Mect Projects”, the product and click on “Choose”.

  1. Insert project name, without blank and special characters but underscore ‘_’.

  2. WARNING: also folder name must not contain spaces.

  3. Select “Next” when asked for “kit selection”.

  4. Select product orientation.

  1. In the “Project Management” window click on “finish”.

WARNING: inside project folder there are the “icons” and “fonts” folders where it is possible to insert its own images / fonts to customize the application.
