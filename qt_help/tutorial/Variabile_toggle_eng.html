Toggle variable

To change state of a toggle variable with a button do the following:

  1. Drag the object on the page.

  2. Double click on button.

  3. Define the status variable StatusVar with name of the Crosstable used to change the value according to the button state. Press “…” key to select the desired variable.

    WARNING: if key changes state only as long as it is pressed, the checkable option must not be selected, otherwise if state must change until next key activation,  the checkable option in the QAbstractButton section must be selected.

WARNING: if the reference variable is changed by PLC, button shows the status of the variable. Then if the variable is forced to 1 button is pressed and execute the linked procedure. If the variable is forced to 0 the button is released.
