Renombrar una página

Está absolutamente prohibido renombrar una página existente para evitar fallos en el funcionamiento del proyecto. Si todavía quiere hacerlo, proceda de la siguiente manera:

  1. Cree una nueva página con el nombre deseado;

  2. En la página que desea renombrar: seleccione todos los elementos. Edit → Select All

  1. Copiar los elementos: Edit → Copy

  2. Vaya a la nueva página creada y seleccione Edit → Paste

  3. Elimine la página que desea eliminar según el procedimiento descrito en el capítulo “Eliminar una página”.
