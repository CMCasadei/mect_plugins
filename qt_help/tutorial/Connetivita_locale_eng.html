Local connection

To download the application from your PC to the Mect device you need to set up your PC with the subnet 5 or change the subnet of the Mect device and set it to the same as your PC.

To do this follow Windows instructions at link: (configure “Internet Protocol Version 4 (TCP/IPv4)”)

https://support.microsoft.com/en-us/help/15089/windows-change-tcp-ip-settings

Follow instructions, including step 7 (press “Advance…”) and select key “Add…” and insert following parameters:

  • IP address: 192.168.5.100 (must be different from operator panel one)

  • Subnet mask: 255.255.255.0


To change the subnet of the Mect device, press the button at the top right (present in the native application of the factory where there is the name of the product purchased) and press: OPTIONS → NETWORK → ETHERNET. Change the IP address and save the change.

To change the subnet in your application, you need to use an “ATCMbutton” and add the name of the “library” page net_conf in the property “pageName”.
