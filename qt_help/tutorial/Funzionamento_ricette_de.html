Rezepte Funktionsweise

Die Rezepte können nur aus den bereits vorhandenen Rezepten auf dem Ziel abgeleitet werden. Neue Familien von Rezepten können am PC erstellt und dann über die im entsprechenden Abschnitt beschriebene Weboberfläche in das Ziel hochgeladen oder direkt aus dem Projekt übernommen werden.

Die Rezepte sind einfache Textdateien im.csv-Format, bei denen jede Zeile die Variable und den zuzuordnenden Wert angibt.

Um beispielsweise ein Rezept zu erstellen, das im ersten Schritt die Ausgabe TPLC_OUT_1 auf 0 und die Ausgabe TPLC_OUT_2 auf 1 setzt und zweiten Schritt die Ausgabe TPLC_OUT_1 auf 1 und die Ausgabe TPLC_OUT_2 auf 0 setzt, schreiben Sie:

TPLC_OUT_1; 0; 1

TPLC_OUT_2; 1; 0

Sie können beliebig viele Schritte (Spalten) hinzufügen.

ACHTUNG: : Verwenden Sie das “;” als Trennzeichen.

Um ein Rezept oder eine Rezeptfamilie in das Gerät zu laden, müssen Sie die Datei/Ordner “zippen” und über die Webschnittstelle (siehen Sie Kapitel “Funktionsweise der Webschnittstelle”) oder direkt mit dem Projekt auf das Gerät übertragen.

ACHTUNG: Der Name der Rezeptdatei darf keine Leerzeichen enthalten.

Im letzteren Fall müssen diese Schritte befolgt werden:

  • Erstellen Sie aus “Ressourcen erkunden” den Ordner “recipe” innerhalb des Projekts.

  • IFügen Sie die.csv-Dateien der Rezepte ein.

  • Klicken Sie mit der rechten Maustaste auf “Add Exising Files..” und wählen Sie die zu ladenden Rezepte aus.

  • Öffnen Sie die Datei mit der Erweiterung “.pro”

  • Geben Sie den folgenden Code ein und ersetzen Sie die Namen der Rezepte durch Ihre eigenen: 

    recipe.files = recipe/Water_Pressure.csv  recipe/Regulator_tank.csv
    recipe.path = /local/data/recipe

    INSTALLS += \
    	recipe

Auf dem Gerät

Die Anzeige/Upload/Bearbeitung der Rezepte wurde in zwei Schritte unterteilt:

  1. Auswahl des anzuzeigenden Rezepts

  1. Anzeigen / Laden / Ändern des ausgewählten Rezeptes

Um die Rezeptseite zu nutzen, verwenden Sie einen ATCMbutton im Seitenwechselmodus und als “pageName” “recipe_select”.

Von diesem Fenster aus können Sie zwischen den Familien der Rezepte und den dazugehörigen Rezepten navigieren.

Nachdem Sie das gewünschte Rezept ausgewählt haben, drücken Sie die Taste, wenn Sie das Rezept oder die Rezeptfamilie auf USB ( ) speichern möchten oder wählen Sie das Rezept aus, um zur Anzeige der aktuellen Rezeptseite zu wechseln.

Die Rezeptdateien sind.csv-Dateien, die aus zwei Spalten bestehen:

  1. Variablenname

  2. Wert der Variablen

Sobald ein Rezept angezeigt wird, können Sie wie folgt vorgehen:

  • Ändern des Wertes einer Variablen: Wählen Sie den Wert aus.

  • Lesen Sie die aktuellen Werte der Variablen, aus denen sich das Rezept zusammensetz(bei allen angeschlossenen Geräten): durch Drücken der Taste Read ( ).

  • Schreiben Sie die im Rezept angezeigten Werte durch Drücken der Taste Load ( ).

Speichern Sie das aktuelle Rezept: Drücken Sie einfach die Schaltfläche Save ( ) und geben Sie den gewünschten Namen ein.
