Compilation et téléchargement de l'application

Une fois le programme terminé, il est nécessaire de compiler le projet pour vérifier l'absence d'erreurs dans le code. Sélectionnez “Build” → “Build All” ou appuyer sur F7.

Une fois compilé, il est nécessaire d'envoyer le projet au panneau opérateur.

Pour transférer le projet nouvellement compilé dans le panneau, procédez comme suit:

  • Sélectionnez l'onglet Target et cliquez avec le bouton droit de la souris sur “TPAC= Resource{localhost}”. Dans la fenêtre qui apparaît, sélectionnez Properties.

  • Définissez l'adresse IP du panneau dans la zone Control Address de l'étiquette TCP/IP, puis cliquez sur OK (par défault : 192.168.5.211 comme pour les panneaux quand ils quittent l'usine)

  • Sélectionnez l'onglet Instance ou Target

  • Sélectionnez TPAC = Resource1 {localhost} et appuyez sur le bouton droit de la souris en sélectionnant Flash (and Download)

  • A la fin du processus, le message “Download Finished” apparaît au bas de la fenêtre.Si une fenêtre d'erreur apparaît, essayez de nouveau de sélectionner “Flash (and Download)”.

  • Si vous ne voulez pas déboguer, il est conseillé de vous déconnecter du panneau, pour ce faire sélectionnez TPAC = Resource1 {localhost} et appuyez sur le bouton droit de la souris, l'option Disconnect
