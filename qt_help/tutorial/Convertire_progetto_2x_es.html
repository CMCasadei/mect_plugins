Siga los siguientes pasajes para convertir un proyecto 2.0.13 en la versión 3.x:

  • Abra su aplicación de PLC desde ATCMControl Engineering y seleccione “File Save Project As..”

  • En “New project name” inserta plc (todo en minuscolas)

  • En “Location” seleccione la ruta del proyecto HMI. En el ejemplo “Termoregolazione” es el nombre del proyecto creado por QtCreator

  • Abra el proyecto HMI à través el icono MectSuite y pulse Build → Run qmake

    Espere hasta que la barra se vuelve verde.

  • Después pulse Build → Clean All

    Espere hasta que la barra se vuelve verde.

  • Pulse Build → Build All.

El proyecto ha sido convertido a 3.x.

ATTENCIÓN:El dispositivo tambien debe ser actualizado. Vea el capítulo “Actualización”
