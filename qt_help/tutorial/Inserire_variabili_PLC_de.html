Einfügen von Variablen in FUP - LD - SFC-Programme

Um die Variablen einzufügen,die wir benötigen, müssen wir das Symbol Insert Var Read in das Zone 2 Fenster ziehen.

Um die gewünschte Variable schnell zuzuordnen, wählen Sie das Objekt unddrücken Sie F4 oder Select Variable mit der rechten Maustaste.

Sie können den ersten Buchstaben der gewünschten Variable eingeben, um einen Filter in der Suche zu haben.

Alternativ können Sie auch auf die im Fenster eingegebene Variable doppelklicken und den Namen ändern, indem Sie die Variable eingeben, der wir sie zuordnen möchten (z.B. eine Variable in der Datei “Crosstable.gvl”) oder die gerade eingefügte Variable auswählen, das Symbol Insert Programming Unit auswählen und auf die Variable doppelklicken, die Sie zuordnen möchten.
