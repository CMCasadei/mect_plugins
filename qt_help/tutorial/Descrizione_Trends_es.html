Descripción Trends

Campo

Descripción

Orientation

Indica la orientación del Trend con respecto al eje temporal:

- “Landscape” → horizontal;

- “Portrait” → vertical.

Visible

Haga que el grafico específico sea visible o no dentro de la página Trends.

Variable

Nombre de la variable que se desea mostrar en el Trend.

Set Color

Color del gráfico relativo que se verá en la página Trends.

Min Value

Valor mínimo a asumir en el Trend en el eje de las ordenadas.

Max Value

Valor máximo a asumir en el Trend en el eje de las ordenadas.

Description

Descripción del gráfico.
