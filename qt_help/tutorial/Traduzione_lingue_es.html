Traducción de idiomas

Para tener la misma interfaz gráfica con traducciones a diferentes idiomas, utilice la herramienta Qt Linguist.

La plantilla de proyecto ya está configurada para que usted tenga los archivos listos para compilar las traducciones al italiano y al inglés.

Para hacer las traducciones, utilice la herramienta Qt linquist que se ejecuta siguiendo la siguiente imagen:

En el menú File → Open , seleccione el archivo languages_*.ts dentro del proyecto del idioma deseado(para que el archivo *.ts esté presente, debe haber compilado su proyecto al menos una vez).

Automáticamente la herramienta propondrá los diferentes campos de texto utilizados en la interfaz, para cada texto que desee traducir, inserte la traducción y presione CTRL e ingrese para confirmar. Una vez finalizadas las traducciones, guarde el archivo y cierre Qt Linguist. Para ver correctamente los idiomas es necesario haber hecho, al menos una vez en el proyecto, “Clean All” y “Run qmake”, ver capítulo “Errores de compilación”.
