Erreur dans le transfert PLC

Si vous ne pouvez plus transférer votre programme PLC vers le panneau , bien que la configuration réseau soit correcte et que ce ne soit pas la première fois que vous effectuez le transfert, vous devez redémarrer le PC.

Pour empêcher le redémarrage de la machine vous pouvez “fermer” l'instance suspendue “4ceng”, via le “Gestionnaire des tâches” de Windows.

Cette anomalie est causée par une fermeture incorrecte du programme qui a laissé une instance du programme en suspens.
