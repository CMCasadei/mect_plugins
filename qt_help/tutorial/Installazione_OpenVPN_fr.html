Installation d'OpenVPN

Installez le client OpenVPN à partir du lien suivant: https://openvpn.net/community-downloads/

A la fin de l'installation, il est nécessaire de copier le certificat, émis par Mect, dans le dossier config (il se trouve dans le dossier d'installation de OpenVPN).

Puis lancez OpenVPN et appuyez connectez pour ouvrir la communication avec votre appareil.
