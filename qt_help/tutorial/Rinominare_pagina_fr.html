Renommer une page

Il est absolument interdit de renommer une page existante pour éviter des dysfonctionnements dans le projet. Si vous voulez toujours le faire, procédez comme suit :

  1. Créez une nouvelle page avec le nom souhaité;

  2. Sur la page que vous souhaitez renommer : sélectionnez tous les éléments. Edit → Select All

  1. Copier des éléments: Edit → Copy

  2. Allez sur la nouvelle page créée et sélectionnez Edit → Paste

  3. Supprimez la page que vous vouliez supprimer en suivant la procédure décrite au chapitre “Supprimer une page”.
