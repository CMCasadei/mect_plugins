Estructura de proyecto vacío

Al abrir un proyecto recién creado, el entorno Qt Creator presenta todos los archivos de proyecto en el panel de proyectos. En concreto, hay una sección dedicada a:

  • common → configuraciones comunes (carga de archivos de idioma)

  • languages → sección de soporte multilingue

  • template → configuraciones genéricas de las diferentes rutas del sistema

  • Headers → Encabezados de origen predeterminados del proyecto (extensión de archivo.h)

  • Sources → Fuentes C++ de los componentes predeterminados del proyecto (extensión de archivo.cpp)

  • Forms → fuentes gráficas en las que se define la apariencia de las páginas (extensión de archivo.ui)

  • Resources → fuentes de iconos e imágenes (sólo archivos.png)

La sección de mayor interés para los usuarios de ATCMPlugin es la que contiene los Form del proyecto, que permite personalizar la apariencia gráfica de las páginas por defecto y, en las páginas definidas por el usuario la definición del comportamiento de los objetos gráficos insertados.
