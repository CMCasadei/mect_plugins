ATCMprogressbar

Object to show a progress bar (or bar graph) linked to a Crosstable integer variable.

To use follows the steps:

  1. Draw object on page.

  2. Double click on object.

  3. Define variable field with Crosstable variable name to read/write. Press “…” button to select desired variable.

  4. Define visibilityVar variable with Crosstable variable to read value to show or not the object. Press “…” button to select desired variable.

  5. Check viewStatus variable to have a graphic feedback for reading/writing.

  6. In the section QProgressBar insert in the properties:

    • orientation: orientation of the bar (vertical or horizontal).

    • minimum: minimum value of the variable.

    • maximum: maximum value of the variable.
