Use of a function block

To use a function block in a program select the block and drag inside the program.

In a program it is possible to use several instances of the function block, so it is necessary to declare each function block.

Select the function block in the program, “doppio click” on the upper part of it (dark side) and write the name of the instance of this function block.



→

Then it is necessary to declare a new instance of the function block (between VAR and END_VAR).

To do this it is possible to drag the function block of the “Class” directly inside VAR and END_VAR and renaming the block (for example “Regolazione_1”).

To insert variables see chapter “Insert variables in FBD – LD – SFC programs”.
