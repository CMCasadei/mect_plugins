Configuration entrées analogiques:

  • 0 non configuré

  • 1 courant

  • 2 tension

  • 3 TCJ (Thermocouple type J)

  • 4 TCK (Thermocouple type K)

  • 5 TCT (Thermocouple type T)

  • 6 PT100E (Résolution d'un degré) plage: -40 +800°C

  • 7 PT100R (Résolution au dixième de degré) plage: -40 +200°C

  • 8 TCS (Thermocouple type S)

  • 9 TCB (Thermocouple type B)

  • 10 TCR (Thermocouple type R)

Ex°: ch2 = PT100E

PLC_AnInConf_2 := 6;
