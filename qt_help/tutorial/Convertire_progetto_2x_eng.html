Convert a 2.0.13 project into 3.x version

The following table shows possible versions configurations for PC and device.

PC Software 3.x

OK

OK

OK

Follows these steps to convert a project 2.0.13 into 3.x version:

  • Open PLC application in ATCMControl Engineering and select “File Save Project As..”

  • In “New project name” insert plc (lowercase)

  • In “Location” select the HMI project path. In the example “Termoregolazione” is the project name created by QtCreator

  • Open HMI project by MectSuite icon and press Build → Run qmake

    Wait until the bar turns green.

  • Then press Build → Clean All

    Wait until the bar turns green.

  • Press Build → Build All.

Project is now converted into 3.x version.

WARNING: it is necessary to update the device also. See “Update” chapter.
