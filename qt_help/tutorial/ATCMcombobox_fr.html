ATCMcombobox

Cet objet permet l'utilisation d'une liste déroulante.

Pour l'utiliser, suivez les étapes ci-dessous:

  1. Faites glisser l'objet sur la page.

  2. Double-cliquez sur l'objet et compilez la table de correspondance en vous rappelant que dans la colonne Value vous devez entrer la valeur qui sera transmise à l'automate(PLC) et dans la colonne Label vous devez entrer la valeur qui sera affichée sur l'interface.

  3. Définissez le champ variable avec le nom de la variable Crosstable à partir de laquelle vous voulez lire/écrire. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  4. Définissez la variable visibilityVar avec le nom de la variable Crosstable à partir de laquelle vous voulez lire la valeur à utiliser pour afficher ou non l'objet. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  5. Cochez la variable viewStatus si vous voulez un feedback graphique sur le succès ou l'échec de l'écriture ou de la relecture.
