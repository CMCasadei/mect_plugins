Fonctionnement des recettes

Les recettes ne peuvent dériver que des recettes préexistantes sur la cible. De nouvelles familles de recettes peuvent être créées par PC puis téléchargées vers la cible via l'interface web décrite dans le paragraphe qui lui est dédié ou être transférées directement depuis le projet.

Les recettes sont de simples fichiers texte au format.csv où chaque ligne spécifie la variable et la valeur à associer.

Par exemple, pour créer une recette qui définit la sortie TPLC_OUT_1 à 0 et la sortie TPLC_OUT_2 à 1 à la première étape et la sortie TPLC_OUT_1 à 1 et la sortie TPLC_OUT_2 à 0 à la deuxième étape, écrire:

TPLC_OUT_1; 0; 1

TPLC_OUT_2; 1; 0

Vous pouvez ajouter autant d'étapes (colonnes) que vous le souhaitez.

ATTENTION: Utilisez le “;” comme séparateur.

Pour charger une recette ou une famille de recettes dans l'appareil, vous devez “zipper” il le fichier/dossier et le transférer dans l'appareil via l'(interface web (voir chapitre “Fonctionnement de l'interface web”) ou directement dans le projet.

ATTENTION: Le nom du fichier de recette ne doit pas contenir d'espaces.

Dans ce dernier cas, ces étapes doivent être suivies:

  • A partir de “explorer les ressources” créer le dossier “recipe” dans le projet.

  • Insérez les fichiers.csv des recettes à l'intérieur.

  • Cliquez avec le bouton droit de la souris sur “Add Exising Files..” et sélectionnez les recettes à charger.

  • Ouvrez le fichier avec l'extension “.pro”

  • Entrez le code suivant et remplacez les noms des recettes par les vôtres: 

    recipe.files = recipe/Water_Pressure.csv  recipe/Regulator_tank.csv
    recipe.path = /local/data/recipe

    INSTALLS += \
    	recipe

Sur l'appareil

L'affichage, le chargement et l'édition des recettes ont été divisés en deux étapes:

  1. Sélection de la recette à afficher

  1. Affichage / chargement / modification de la recette sélectionnée

Pour utiliser la page de recette, utilisez un ATCMbutton en mode changement de page et comme “pageName” utilisez “recipe_select”.

A partir de cette fenêtre, vous pouvez naviguer entre les familles de recettes et les recettes qui leur appartiennent.

Après avoir sélectionné la recette désirée, appuyez sur la touche si vous voulez sauvegarder la recette ou la famille de recettes sur USB ( ) ou sélectionnez la recette pour passer à l'affichage de la page de recette actuelle.

Les fichiers de recettes sont des fichiers.csv constitués de deux colonnes:

  1. Nom de variable

  2. Valeur de la variable

Une fois qu'une recette est affichée, vous pouvez procéder comme suit:

  • Modifier la valeur d'une variable : sélectionner la valeur.

  • Lire les valeurs actuelles des variables qui composent la recette ((sur tous les instruments connectés): en appuyant sur le bouton Read ( ).

  • Ecrire sur les appareils les valeurs affichées dans la recette en appuyant sur le bouton Load ( ).

Enregistrer la recette actuelle : appuyez simplement sur le bouton Save ( ) et tapez le nom désiré.
