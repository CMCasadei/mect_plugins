Traduction des langues

Afin d'avoir la même interface graphique avec des traductions dans différentes langues, utilisez l'outil Qt Linguist.

Le modèle de projet est déjà configuré de sorte que vous avez déjà les fichiers prêts à compiler les traductions en italien et en anglais.

Pour faire les traductions, utilisez l'outil Qt linquist que vous exécutez en suivant l'image suivante:

Dans le menu File → Open, choisissez le fichier languages_*.ts dans le projet de la langue souhaitée(pour que le fichier *.ts soit présent, vous devez au moins avoir compilé votre projet une fois).

L'outil proposera automatiquement les différents champs de texte utilisés dans l'interface, pour chaque texte que vous voulez traduire insérez la traduction et appuyez sur CTRL et entrez pour confirmer. Une fois les traductions terminées, enregistrez le fichier et fermez Qt Linguis. Pour voir les langues correctement vous devez avoir fait, au moins une fois dans le projet, “Clean All” et “Run qmake”, voir chapitre “Erreurs de compilation”.
