Eine neue Sprachdatei hinzuzufügen

Um eine neue Sprachdatei hinzuzufügen, müssen Sie:

  • Gehen Sie in den Projektbaum unter der Rubrik Sprachen.

  • Öffnen Sie die Datei “languages.pri” und fügen Sie den Namen der Übersetzungsdatei der gewünschten Sprache hinzu, wobei Sie daran denken, dass die Datei languages_*.ts heißen muss, wo der Code der gewünschten Sprache anstelle von * platziert werden muss (z.B: fr → Französisch). Die zu verwendenden Codes sind auf dieser Webseite sichtbar: wiki.freepascal.org/Language_Codes

  • ACHTUNG:zwischen dem Namen einer Datei und der anderen müssen Sie das Symbol “ \ ” einfügen.

  • Kompilieren, um die Übersetzungsdatei auf dem Dateisystem zu erstellen (Erstellung von Dateien: *.qm und *.ts).

  • Gehen Sie zum Projektbaum unter dem Abschnitt “common” → “Resources”.

  • Fügen Sie zur Ressourcendatei “languages.qrc” die entsprechende *.qm-Datei hinzu, die durch Kompilieren des vorherigen Schrittes erstellt wurde.

  • Alle speichern: File → Save “resources.qrc”.
