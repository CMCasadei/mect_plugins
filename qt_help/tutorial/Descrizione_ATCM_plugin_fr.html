Description ATCM plugin

Les objets graphiques qui peuvent être utilisés sur une page de projet sont situés dans la section plugin ATCM et sont:

  • ATCMbutton

  • ATCMcombobox

  • ATCMspinbox

  • ATCMlabel

  • ATCMprogressbar

  • ATCMslider

  • ATCMled

  • ATCManimation

  • ATCMtime

  • ATCMdate

  • ATCMgraph

Les ATCMPlugin intègrent une série de propriétés graphiques dont l'utilisation vous permet de créer facilement une application IHM sans écrire de code.
