ATCMcombobox

Este objeto permite el uso de un combobox.

Para utilizarlo, siga los pasos siguientes:

  1. Arrastre el objeto a la página.

  2. Haga doble clic sobre el objeto y compile la tabla de mapeo recordando que en la columna Value debe introducir el valor que se pasará al PLC y en la columna Label debe introducir el valor que se mostrará en la interfaz.

  3. Defina el campo variable con el nombre de la variable Crosstable de la que desea leer/escribir. Pulse el botón “…” para seleccionar la variable deseada.

  4. Definire la variabile visibilityVar con el nombre de la variable Crosstable de la que se desea leer el valor a utilizar para mostrar o no el objeto. Pulse el botón “…” para seleccionar la variable deseada.

  5. Marque la variable viewStatus si desea obtener información gráfica sobre el éxito o el fracaso de la escritura o la relectura.
