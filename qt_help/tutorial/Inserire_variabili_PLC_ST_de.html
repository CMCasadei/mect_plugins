Einfügen von Variablen in ST-Programme

Um Variablen schnell einzufügen, positionieren Sie den Cursor auf die gewünschte Zeile und drücken Sie F4 oder Select Variable mit der rechten Maustaste.

Sie können den ersten Buchstaben der gewünschten Variable eingeben, um einen Filter in der Suche zu haben.

Um Variablen, Funktionen oder Funktionsblöcke in das Programm einzufügen, folgen Sie alternativ die folgenden Schritte:

  • Wählen Sie “View” → “POU/Variable Selection” → “Variable Selection&rdquo (oder drücken Sie die rechte Maustaste innerhalb eines Programms);

    • Wählen Sie die gewünschte Variable (Sie können auch Filter verwenden) und drücken Sie “Apply”

Um eine Funktion / einen Funktionsblock einzufügen:

  • Wählen Sie “View” → “POU/Variable Selection” → “POU Selection”

    • Wählen Sie die gewünschte Funktion / den gewünschten Funktionsblock aus und drücken Sie “Apply”

Natürlich können die Namen von Variablen, Funktionen und Funktionsblöcken direkt geschrieben werden, ohne die oben beschriebene Vorgehensweise zu befolgen.
