Password operating principle

All systems have default password equal to “0” for all 3 levels.

As default passwords are the same for the 3 levels, first login has highest priority. So first login is assigned to “Admin” and the ‘Password’ key modifies the “Admin” password.

Next login with “0” password is assigned to “SuperUser” and it is possible to modify its password. Next login with “0” password is assigned to “User” and it is possible to modify its password.

Last login with “0” password is assigned to “Operator”.
