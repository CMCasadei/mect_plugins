Date/time reading by HMI function

It is possible to use date and time with some function in the HMI environment. These values can be stored in Crosstable variables to use inside PLC program. For example insert the “day” and “minutes” variables in the Crosstable.

First include in “automation.cpp” file:

#include <QDate>

#include <QTime>

Inside:

void loop (void)

insert:

doWrite_giorno(QDate::currentDate().day());

doWrite_minuti(QTime::currentTime().minute());

In the variables “giorno” and “minuti” there are the values of actual day and minutes and it is possible to use it in the PLC.

To use the function for month, year, etc see: doc.qt.io/qt-4.8/qdate

To use the function for hour, minutes and seconds see: doc.qt.io/qt-4.8/qtime
