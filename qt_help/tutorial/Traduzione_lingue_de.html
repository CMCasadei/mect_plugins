Sprachübersetzung

Um die gleiche grafische Oberfläche mit Übersetzungen in verschiedene Sprachen zu haben, verwenden Sie das Tool Qt Linguist.

Die Projektvorlage ist bereits so konfiguriert, dass Sie die Dateien zur Erstellung von Übersetzungen ins Italienische und Englische bereits zur Verfügung haben.

Um die Übersetzungen durchzuführen, verwenden Sie das Qt Linquist-Tool, das Sie nach dem folgenden Bild ausführen:

Wählen Sie aus dem Menü File → Open die Datei languages_*.ts innerhalb des Projekts der gewünschten Sprache (damit die *.ts-Datei vorhanden ist, müssen Sie Ihr Projekt mindestens einmal kompiliert haben).

Automatisch schlägt das Tool die verschiedenen Textfelder vor, die in der Benutzeroberfläche verwendet werden, für jeden Text, den Sie übersetzen möchten, fügen Sie die Übersetzung ein und drücken STRG und Enter zur Bestätigung. Sobald die Übersetzungen abgeschlossen sind, speichern Sie die Datei und schließen Sie Qt Linguist. Um die Sprachen korrekt anzuzeigen, müssen Sie mindestens einmal im Projekt, “Clean All” und “Run qmake”, siehen Sie Kapitel “ Kompilierungsfehler”.
