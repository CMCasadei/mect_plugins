Umschaltvariable

Um die Schaltfläche zu verwenden, um einen Zustandswechsel einer Umschaltvariablen vorzunehmen, führen Sie die folgenden Schritte aus:

  1. Ziehen Sie das Objekt auf die Seite.

  2. Doppelklicken Sie auf die Schaltfläche.

  3. Definieren Sie die Statusvariable StatusVar mit dem Namen der Crosstable-Variablen, mit der Sie den Wert in Abhängigkeit vom Status der Schaltfläche ändern möchten. Drücken Sie die Taste “…”,um die gewünschte Variable auszuwählen.

    HINWEIS: Wenn Sie den Status nur so lange ändern möchten, bis die Taste gedrückt wird, überprüfen Sie, ob die Option checkable NICHT angekreuzt wird; wenn Sie den Status bis zum nächsten Tastendruck ändern möchte, überprüfen Sie, ob die Option checkable im Abschnitt QAbstractButton angekreuzt wird.

HINWEIS: Wenn sich der Wert der Führungsgröße von der SPS aus ändert, übernimmt die Taste den Status der Variablen selbst. Wenn die Variable auf 1 gesetzt wird, wird die Taste gedrückt und führt alle zugehörigen Operationen aus. Wenn die Variable auf 0 gesetzt wird, wird die Taste losgelassen.
