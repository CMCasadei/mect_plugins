Polices

Les polices suivantes sont installées lors de l'installation du paquet MectSuite:

  • DejaVu Sans

  • DejaVu Serif

Dans le développement de vos applications, il est fortement recommandé d'utiliser l'une de ces polices car tout autre type de police utilisé dans QtCreator ne s'affichera pas correctement lorsque vous transférerez l'application dans le panneau.

Si vous voulez ajouter une police spécifique sur la cible, vous devez l'inclure dans le projet, en suivant la procédure du chapitre “Ajouter des images / polices”.

ATTENTION: il est possible d'insérer uniquement des polices avec l'extension .ttf
