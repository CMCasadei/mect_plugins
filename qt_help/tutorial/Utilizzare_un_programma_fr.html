Utiliser un programme

A partir de l'onglet “Class” il est possible d'étendre “Programs” où vous pouvez trouver les programmes par défaut écrits en langage ST.

Si nous double-cliquons sur un programme, nous voyons que sur le côté droit s'ouvrira une fenêtre dans laquelle insérer le code ST pour effectuer l'automatisation.

Si nous devons utiliser des variables locales, nous devons les déclarer dans l'espace entre VAR et END_VAR, par exemple: pippo : bool;

Une fois déclarée la variable locale nous pouvons l'utiliser à l'intérieur de notre programme, tandis que pour les variables globales (celles qui sont à l'intérieur du fichier “Crosstable.gvl” et qui ont été générées par le “Crosstable Editor”) peuvent être utilisées directement dans tout programme, en insérant le nom de la variable que vous voulez utiliser.
