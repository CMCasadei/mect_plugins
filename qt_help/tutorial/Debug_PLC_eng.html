Debug PLC

At the end of the application “Download” and at the restore of communication, the “Debug” mode is automatically started.

In Debug mode a tab appears for the programs running on target (see red circles) and in the tab “Instance” the current status of task is shown (active, stop, …).

Circles of targets are “green” if connected and “yellow” if not connected.

It is possible to enable the variables update of the running program dragging the variable in the “watch window”, as in the blue arrow in figure, or in menu “View → Watch”.

Variables update appears also “on the fly” passing with mouse on the variable (can be fixed with popup → Set Watchpoint).

All commands to debug are also in menu “Debug”.

WARNING: it is not possible to debug a function but it is possible to debug a function block.
