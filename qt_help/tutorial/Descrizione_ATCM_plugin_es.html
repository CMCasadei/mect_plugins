Descripción ATCM plugin

Los objetos gráficos que se pueden utilizar en una página de proyecto se encuentran en la sección ATCM plugin y son:

  • ATCMbutton

  • ATCMcombobox

  • ATCMspinbox

  • ATCMlabel

  • ATCMprogressbar

  • ATCMslider

  • ATCMled

  • ATCManimation

  • ATCMtime

  • ATCMdate

  • ATCMgraph

Los ATCMPlugin integran una serie de propiedades gráficas cuyo uso permite crear fácilmente una aplicación HMI sin necesidad de escribir código.
