Utiliser une fonction

Pour utiliser une fonction dans un programme, sélectionnez-la dans l'onglet “Class” et faites-la glisser dans le programme.

Pour l'insertion des variables, voir le chapitre “Insérer des variables dans les programmes FBD – LD – SFC”.

En particulier, cette fonction permet de reconditionner les valeurs d'un signal avec celles souhaitées. Par exemple, le signal d'entrée sera un signal de tension analogique (PLC_AnIn_1) avec une plage de 0-10V (ISI - FSI) et vous voulez reconditionner ce signal pour avoir une limite inférieure/supérieure de -200+300 (ISL-FSL).

La variable “Valore_ricondizionato” prend une valeur entre -200 e +300, selon la valeur de la variable (PLC_AnIn_1).
