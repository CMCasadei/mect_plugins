Übertragung der Anwendung auf das Gerät in DEBUG

Sie können die HMI-Anwendung im Debug-Modus starten und alles im Debug-Modus kompilieren (siehen Sie roter Kreis); wählen Sie das MECT-Symbol und wählen Sie den gewünschten Modus.

Um den Haltepunkt in eine Zeile einzufügen, klicken Sie im grauen Teil links neben der Quellzeilennummer: z.B. in Sources → page100.cpp zur page::page100() Funktion.

Um die Sequenz “Build, Deploy und Debug” zu starten, drücken Sie einfach F5 oder das “Grüne Dreieck mit Marienkäfer” (siehen Sie blauer Kreis).

Unter Debug (gelber Kreis) können Sie Ihre Anwendung im “Debug”-Modus kontrollieren.
