Variable de conmutación

Para usar el botón para hacer un cambio de estado de una variable de conmutación, siga los pasos siguientes:

  1. Arrastre el objeto a la página.

  2. Haga doble clic en el botón.

  3. Defina la variable de estado StatusVar con el nombre de la variable Crosstable que desea utilizar para cambiar el valor en función del estado del botón. Pulse el botón “…” para seleccionar la variable deseada.

    NOTA: si desea cambiar el estado sólo hasta que se pulse el botón, compruebe que la opción checkable NON ESTE marcada; si desea cambiar el estado hasta la próxima vez que pulse el botón, compruebe que la opción checkable en la sección QAbstractButton ESTE marcada.

NOTA: si el valor de la variable de referencia cambia desde el PLC, el pulsador asumirá el estado de la propia variable. Si la variable es forzada a 1 entonces el botón será presionado y realizará cualquier operación asociada. Si la variable es forzada a 0 entonces se suelta el botón.
