Using a function

To use a function in a program select it in the “Class” tab and drag it inside the program.

To insert variables see chapter “Insert variables in FBD – LD – SFC programs”.

This function allows to scale an input to a desired output. For example the input signal is an analogue voltage input (PLC_AnIn_1) 0-10V (ISI and FSI) and it is scaled to -200+300 (ISL-FSL).

The “Valore_ricondizionato” variable has now value between -200 and +300 according to input value (PLC_AnIn_1).
