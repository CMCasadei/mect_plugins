PLC

To write PLC automation ATCM control is used.

The PLC software consists of a series of cycles called “Task” which are guaranteed to be executed at pre-established times.

Each task can contain a “program” or more of them, in which the programmer adds his own code with the preferred language. Within the programs can be used other blocks of recurring code called “functions” and / or “function blocks”.

Unlike the functions, the function blocks have these main features:

  • debugging available;

  • use of timers;

  • multiple inputs and outputs.

It is up to the programmer to choose their use within his own programs.



On the device an binary code and not a source code of the PLC program is downloaded. For this reason it is not possible to retrieve the source code of the product from the device.

WARNING: PLC project must:

  • Be named as plc

  • Be inside the project folder created by HMI environment (same level as “config” folder).
