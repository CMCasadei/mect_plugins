Eine Seite Umbenennen

Es ist absolut verboten, eine bestehende Seite umzubenennen, um Fehlfunktionen im Projekt zu vermeiden. Wenn Sie es trotzdem tun wollen, gehen Sie wie folgt vor:

  1. Erstellen Sie eine neue Seite mit dem gewünschten Namen;

  2. Auf der Seite, die Sie umbenennen möchten: Alle Elemente auswählen. Edit → Select All

  1. Elemente kopieren: Edit → Copy

  2. Gehen Sie auf die neu erstellte Seite und wählen Sie Edit → Paste

  3. Entfernen Sie die Seite, die Sie entfernen möchten, wie im Kapitel “Eine Seite entfernen” beschrieben.
