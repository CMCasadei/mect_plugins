ATCMPlugin description

Graphics object to use directly in a project page are in the ATCM plugin section and they are:

  • ATCMbutton

  • ATCMcombobox

  • ATCMspinbox

  • ATCMlabel

  • ATCMprogressbar

  • ATCMslider

  • ATCMled

  • ATCManimation

  • ATCMtime

  • ATCMdate

  • ATCMgraph

ATCMPlugin integrates a series of graphical property that allows a simple realization of an HMI interface without writing code.
