Trends

Les trends sont des graphiques historisés et sont décrites par une série de fichiers de configuration qui peuvent être modifiés par l'utilisateur, avec la possibilité pour ce dernier d'en ajouter de nouveaux.

ATTENTION: Pour pouvoir afficher les tendances, les variables de référence doivent être historisées (voir chapitre “Fonctionnement de l'historique”) et doivent avoir été activées par la fonction logStart().
