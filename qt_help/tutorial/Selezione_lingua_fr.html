Sélectionner la langue depuis l'appareil

La sélection de la langue sur l'appareil peut se faire de deux manières différentes:

  1. A travers les pages par défaut:

A partir de la page de menu, sélectionnez “OPTIONS” → “LANGUAGE”

  1. A travers d'une page personnalisée:

Sélectionnez un ATCMbutton et avec le bouton droit de la souris, sélectionnez Aller à Go to slot… dans le menu qui apparaît ; puis sélectionnez la ligne pressed() et appuyez sur OK.

La fenêtre suivante peut apparaître:

Si tel est le cas, les étapes suivantes doivent être effectuées:

  • Faire le Build du projet

  • Fermer et rouvrir le proje

Puis une page avec du code C+++ apparaîtra:

void page100::on_atcmButton_3_pressed()

{

}


Insérez le code suivant dans les crochets bouclés de la fonction ci-dessus :

if(translator->load(QString(":/translations/languages_it.qm"))) {

QSettings settings(CONFIG_FILE,QSettings::IniFormat);

settings.setValue("SYSTEM/language","it");

settings.sync();

}

else

{

QMessageBox::information(this,trUtf8("errore"),trUtf8("Caricamento lingua"));

}


Répétez ces étapes pour toutes les sélections et ne modifiez que les parties écrites en vert de façon appropriée.

Pour l'anglais, remplacer: :/translations/languages_en.qm

Pour le français, remplacer: :/translations/languages_fr.qm

ATTENTION: il est nécessaire d'insérer dans la partie initiale du fichier la ligne suivante: #include <QSettings>
