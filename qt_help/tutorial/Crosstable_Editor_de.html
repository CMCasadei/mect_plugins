Crosstable Editor

Nachdem Sie das Projekt in Qt erstellt haben, müssen Sie zuerst die Variablen definieren,die Sie für Ihre Anwendung verwenden möchten. Dazu verwenden wir die "Crosstable", die die Tabelle der globalen Variablen generiert, die gemeinsam von der HMI-Umgebung und der SPS verwendet wird.

Um die “Crosstable” zu öffnen,klicken Sie auf das Symbol von MectSuite:

Hier ist die “Crosstable”:

Im Abschnitt “Goto Section” (siehen Sie gelbes Rechteck) können Sie den Typ der Variablen auswählen, die in der folgenden Tabelle angezeigt werden sollen.

Der rot markierte Bereich ist der Teil, den der Benutzer definieren kann, um seine eigenen Projektvariablen zu erstellen und zu bearbeiten.

Wenn Sie mit der Maus über jedes Feld fahren, können Sie eine kurze Beschreibung sehen..

So fügen Sie eine Variable ein oder ändern sie:

  1. Wählen Sie die Zeile aus der Tabelle aus;

  2. Füllen / ändern Sie die Felder im ausgewählten Bereich mit rot aus;

  3. Wählen Sie die Taste “Enter” auf der Tastatur oder eine beliebige Zeile in der Tabelle, um die neu erstellte / geänderte Variable einzufügen.

HINWEIS: Innerhalb der Tabelle ist es möglich, mit den Pfeilen zu navigieren, als wäre es eine Tabellenkalkulation.
