Languages translation

To have the same graphic interface but different languages use tool Qt Linguist.

The project template is already configured to have the italian and english files ready to compile.

To translate use Qt Linquist and to run it follow the figure:

In the menu File → Open choose file languages_*.ts in the project of the desired language (the file is present if the project has been compiled at least once).

Automatically the tool propose various text fields used in the interface, insert the translation and press CTRL and enter to confirm. At the end of translations save the file and close Qt Linguist. To see correctly the languages you must have done at least once in the project, “Clean All” and “Run qmake”, see chapter “Build errors”.
