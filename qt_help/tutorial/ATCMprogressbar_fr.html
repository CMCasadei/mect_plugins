ATCMprogressbar

Cet objet permet l'affichage d'une barre de progression (ou bargraphe) liée au contenu d'une variable entière de la Crosstable.

Pour l'utiliser, suivez les étapes ci-dessous:

  1. Faites glisser l'objet sur la page.

  2. Double-cliquez sur l'objet.

  3. Définissez le champ variable avec le nom de la variable Crosstable à partir de laquelle vous voulez lire/écrire. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  4. Définissez la variable visibilityVar avec le nom de la variable Crosstable à partir de laquelle vous voulez lire la valeur à utiliser pour afficher ou non l'objet. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  5. “Cochez“ la variable viewStatus au cas où vous voulez un retour graphique sur le succès ou l'échec de l'écriture ou de la relecture.

  6. Dans la section QProgressBar entrez dans la propriété:

    • orientation: l'orientation de la barre (verticale ou horizontale).

    • minimum: la valeur minimale que la variable va prendre.

    • maximum: la valeur maximale que la variable prendra.
