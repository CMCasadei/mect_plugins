Importation d'un bloc fonction / d'une fonction

Il est possible d'importer une fonction/ un bloc fonction précédemment utilisé dans un autre projet. Pour ce faire, vous devez:

  • Sélectionner l'onglet “File”

  • Sélectionnez “Source Files” et appuyer sur le bouton droit de la souris

  • Sélectionner “Add File(s)”

  • Sélectionner la fonction/ le bloc fonction que vous souhaitez importer dans le nouveau projet

  • Sélectionner “Open”

La fonction/ le bloc fonction qui vient d'être importé sera visible dans l'onglet “Class” sous “Functions” ou “Function Blocks”. Dans ce cas, une fonction a été importée.
