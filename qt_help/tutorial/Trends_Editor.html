Trends Editor

Con il “Trends Editor” è possibile configurare le variabili storicizzate (variabili che hanno il parametro S, F, V e X nel campo “Update” del “Crosstable Editor”) di cui si vuole avere il grafico in funzione del tempo.

Nel “Trend Editor” si possono impostare 4 tracce (grafici).

All’apertura dell’editor si avrà un avviso del fatto che le variabili presenti nel campo “Nome variabile” non sono presenti nella Crosstable. Questo avviso si avrà ogni qualvolta ci sia una non congruenza tra le variabili della Crosstable e quelle salvate nel file di Trend.

In un progetto è possibile avere più file di Trend; in questo caso all’avvio del “Trends Editor” viene richiesto di selezionare il file di Trend che si desidera vedere/modificare.
