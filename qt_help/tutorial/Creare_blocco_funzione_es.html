Crear un bloque de función

Proceda del siguiente modo para crear un bloque de función, por ejemplo, para el lenguaje Function Block Diagram (FBD):

  • Seleccione Class

  • Seleccione Function Blocks en el árbol y pulse el botón derecho del ratón

  • Seleccione y pulse Insert Function Block

  • En la ventana que aparece, seleccione Function Block Diagram FB

  • Introduzca el nombre que desea dar al bloque de función (por ejemplo: RegolazioneON_OFF)

  • Pulse OK

El bloque de función fue creado de esta manera, de hecho expandiendo la rama Function Blocks del árbol de proyecto,vemos la rama.

Para entrar en el bloque de función, seleccione y haga doble clic en el nombre del bloque de función. La ventana que aparece consta de dos zonas:

  • Zona 1 para la definición de variables

  • Zona 2 de Código

Como primer paso, hacer que el bloque de función vea las variables definidas en Crosstable. Para ello, sitúese con el ratón en la zona 1 (que cambia el color de fondo), inmediatamente después del nombre del bloque de función, e introduzca la escritura: #import “Crosstable.gvl” como puede ver en la figura.

Ahora vaya a la zona 2 para introducir el código de programa.

Para insertar operadores seleccione el icono Insert Programming Unit donde aparecerá la ventana en la que puede elegir los operadores de la biblioteca. Seleccionando y arrastrando el operador necesario (por ejemplo, el operador ADD que suma dos variables), llévelo a la zona 2.

Para conectar las variables al operador seleccione el icono, en este punto simplemente haga las conexiones como en un editor normal para diagramas de cableado. Para la inserción de variables ver el capítulo variables ver el capítulo “Insertar variables en programas FBD – LD – SFC”.
