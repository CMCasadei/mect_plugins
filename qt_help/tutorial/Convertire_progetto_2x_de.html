Folgen Sie den folgenden Schritten, um ein 2.0.13-Projekt in eine 3.x-Version zu konvertieren:

  • Öffnen Sie Ihre SPS-Anwendung von ATCMControl Engineering und wählen Sie “File Save Project As..”

  • In “New project name” einfügen Sie plc (alles Kleinbuchstaben)

  • In “Location” Wählen Sie den Pfad des HMI-Projekts aus. Im Beispiel “Termoregolazione” ist der Projektname mit QtCreator erstellt.

  • Öffnen Sie das HMI-Projekt durch MectSuite Symbol und Drücken Sie Build → Run qmake

    Warten Sie, bis die Task-Leiste grün wird.

  • Denn Drücken Sie Build → Clean All

    Warten Sie, bis die Task-Leiste grün wird.

  • Drücken Sie Build → Build All.

Das Projekt wurde in 3.x konvertiert.

ACHTUNG: Das Gerät muss ebenfalls aktualisiert werden. Siehen Sie das kapitel “Aktualisierung”
