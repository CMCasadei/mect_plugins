Modification du rétroéclairage via la fonction HMI

Les fonctions suivantes peuvent être utilisées pour modifier la luminosité du rétroéclairage du moniteur via le code HMI:

  • set_backlight_level(valeur) /*définit la valeur*/

  • valeur = get_backlight_level() /*obtient la valeur*/

La valeur de luminosité est un nombre compris entre 0 et 100.
