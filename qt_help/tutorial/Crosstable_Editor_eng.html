Crosstable Editor

After project creation in Qt, first of all define variables used in the application. To do this use “Crosstable Editor” that generates a global variables table that is shared between HMI and PLC.

To start “Crosstable Editor” see the following figure:

Here the “Crosstable Editor”:

The red zone is where the user creates and modify the project variables.

Moving mouse over each field a brief description is shown.

In the below table the parameters of the red section are automatically inserted.

Filling the table can be done in 3 ways:

  1. Depending on variable type (see yellow box) Retentive or not, the program insert new values in the first empty row;

  2. By means of key “Insert a row”, specifying destination row;

  3. By means of key “previous block”, if the new variable must be inserted in the block of variables already created.

Press key “Update table” to update the table.
