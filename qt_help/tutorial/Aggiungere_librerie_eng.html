Add libraries to the project

It is possible to add libraries to the project. In the “Class” tab, select with right mouse key “Libraries” item and press “Add library”. Then select the library (.4cl extension) and press “Open”.

WARNING: on TPAC and TP devices it is possible to import only the libraries:

  • ATCM_File.4cl

  • ATCM_System.4cl

  • ATCM_System2.4cl

  • ATCM_Utility.4cl
