Transférer l'application à l'appareil en DEBUG

Vous pouvez lancer l'application IHM en mode débogage, en compilant tout en mode débogage (voir cercle rouge); sélectionnez l'icône MECT et choisissez le mode que vous voulez.

Pour insérer le point d'arrêt dans une ligne, cliquez dans la partie grise à gauche du numéro de ligne source: par exemple dans Sources → page100.cpp sur la fonction page::page100().

Pour lancer la séquence “Build, Deploy et Debug”, il suffit d'appuyer sur F5 ou sur le “Triangle vert avec coccinelle” (voir cercle bleu).

Sous Debug (cercle jaune) vous pouvez contrôler votre application en mode “Debug”.
