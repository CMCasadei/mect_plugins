Help de ATCMcontrol

Pour plus de détails sur l'utilisation des différents langages de programmation d'ATCMcontrol, il existe une aide en ligne (en anglais) qui explique comment utiliser les différents éléments et opérateurs.

Pour accéder à ATCMControl Help à partir du menu sélectionnez: Help → ATCMControl Engeineering Help → Help Topics

Dans la fenêtre qui apparaît, sélectionnez “Index” et écrivez “FBD”
