Installation

To develop a graphic interface you need to download the following programs:

  • QtCreator and plugin: install the QtCreator development suite for the QT4.8.5 libraries and the plugin that are useful for the HMI programming.

  • ATCMcontrol Engineering: PLC tools

  • CanBuilder: CAN network tools.

This programs have been tested on Windows 7, 8, 8.1 and 10, previous Windows versions are not supported.

For more information see “Installation” folder in “MectSuite_xy”.
