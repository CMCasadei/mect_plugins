Insérer un programme dans une Tâche

Après avoir créé le programme, nous pouvons l'insérer dans la tâche que nous voulons. Pour ce faire, sélectionnez le programme dans l'onglet “Class”, et faites-le glisser (avec le bouton gauche de la souris) dans la tâche souhaitée.

REMARQUE: Le même programme peut être utilisé puis instancié dans plusieurs tâches.

Dans le programme, il est possible d'insérer et d'utiliser une fonction et/ou un bloc fonction ; voir l'aide ATCMcontrol pour plus de détails.
