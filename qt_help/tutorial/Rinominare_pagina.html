Rinominare una pagina

E’ assolutamente vietato rinominare una pagina esistente onde evitare malfunzionamenti nel progetto. Se lo si volesse fare ugualmente procedere nel seguente modo:

  1. Creare una nuova pagina con il nome desiderato;

  2. Nella pagina che si vorrebbe rinominare: selezionare tutti gli elementi. Edit → Select All

  1. Copiare gli elementi: Edit → Copy

  2. Andare sulla nuova pagina creata e selezionare Edit → Paste

  3. Rimuovere la pagina che si voleva rimuovere secondo la procedura del capitolo “Rimuovere una pagina”.
