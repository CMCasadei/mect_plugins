Ajouter un nouveau fichier de langues

Pour ajouter un nouveau fichier de langue, vous devez:

  • Aller à l'arborescence du projet sous la section des langues.

  • Ouvrez le fichier “languages.pri” et ajoutez le nom du fichier de traduction de la langue désirée, en vous rappelant que le fichier doit s'appeler languages_*.ts où le code de la langue désirée doit être placé au lieu de * dovrà (ex: fr → français). Les codes à utiliser sont visibles sur cette page web: wiki.freepascal.org/Language_Codes

  • ATTENTION: entre le nom d'un fichier et l'autre vous devez insérer le symbole “ \ ”.

  • Compiler afin de créer le fichier de traduction sur le système de fichiers (création des fichiers : *.qm et *.ts).

  • Allez dans l'arborescence du projet sous la rubrique “common” → “Resources”.

  • Ajoutez au fichier de ressources “languages.qrc” le fichier *.qm correspondant créé en compilant l'étape précédente.

  • Enregistrer tout: File → Save “resources.qrc”.
