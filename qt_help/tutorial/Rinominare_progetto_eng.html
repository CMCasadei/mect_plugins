Rename a project

To rename the project folder created by QtCreator do the following:

  1. Close project to rename.

  2. Rename project folder (don't use special characters or blanks except _).

  3. Delete in the project folder the file with “.pro.user” extension.

  4. Rename file with “.pro” extension with the name you choose in the point 2.

  5. Rename file with “.pro.shared” extension with the name you choose in the point 2.

  6. Open new project in QtCreator.

  7. When you open the project the following image appears, then press “Configure Project” key.

  1. Perform “Clean All” and “Run qmake” (see chapter “Build errors”)

  2. Build the project.

  3. Set the parameters to send the project to the target, see chapter “Download to the panel of a graphic application”.
